What's GitHub

Git: is a program installed on your machine. It tracks the changes you make to your project files (such as code). It allows you to save "snapshots" of your project over time, including previous versions, and see who changed what and when. GitHub: is a cloud hosting service (like Google Drive or Dropbox, but for programmers). Unlike saving a version of your project (a Git repository) to your local machine, GitHub allows you to upload it to the internet.
